# PHP_COMPOSER_INSTALLATION

   **An automated Installation shell script for php-composer  for linux ubuntu OS's.**

* Download the script.

* change the file permission with 777 
   
    ` ` ` sudo chmod 777 composer.sh ` ` `

* Run the script with sudo.  
    
    ` ` ` sudo ./composer.sh ` ` `
    
## Prerequisite:

  * Ubuntu OS.
  * php
